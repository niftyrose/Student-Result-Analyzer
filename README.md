# Student Result Analyzer

A Python-based data analysis project that processes student academic records, calculates total marks, percentages, and grades, and generates performance insights using Pandas.

## Features

* Read student data from CSV files
* Calculate total marks automatically
* Compute percentage scores
* Assign grades based on performance
* Analyze student results efficiently
* Perform data processing using Pandas

## Technologies Used

* Python
* Pandas
* CSV File Handling
* Data Analysis

## Sample Analysis

The system:

1. Reads student marks from a CSV file.
2. Calculates total marks.
3. Computes percentage scores.
4. Assigns grades based on predefined criteria.
5. Displays the final analyzed dataset.
